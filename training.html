ESD Training Modules

A structured pathway for clinicians, educators, and helpers who want to work skillfully with empathy dysregulation across the spectrum instead of chasing isolated symptoms.

How the training is structured

The ESD training sequence moves from understanding and mapping empathy patterns to applying the framework in real cases. Each module includes teaching content, reflection, and practical tools you can use the same day.

Module 1

Foundations of Empathy Spectrum Dysregulation

Learn the core architecture of ESD, the six subtype profiles, and how empathy dysregulation emerges across development instead of appearing out of nowhere in adulthood.

  • Differentiate affective, cognitive, and somatic empathy
  • Understand the six ESD subtype profiles
  • Connect empathy patterns to developmental and relational context
Format: video + slides + workbook Level: foundational
View module (coming soon) Download outline (PDF)
Module 2

Assessing Empathy Profiles in Practice

Work with the ESD toolkit to map empathy patterns, defensive styles, and nervous system responses so you can see what is actually happening under the diagnostic surface.

  • Use ESD assessment questions and observation prompts
  • Identify hyper, hypo, oscillating, and dark empathy patterns
  • Translate empathy patterns into a clear case map
Format: case demos + worksheets Level: intermediate
View module (coming soon) Explore toolkit
Module 3

Intervention Pathways and Case Formulations

Integrate ESD with attachment work, nervous system regulation, and practical boundaries so you have a clear path for both prosocial and high risk empathy profiles.

  • Match intervention style to subtype presentation
  • Support hyperempathic clients without burning them out
  • Address shutdown and dark empathy patterns safely and clearly
Format: frameworks + case examples Level: advanced
